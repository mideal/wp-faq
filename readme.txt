== Description ==

Question answer, ajax, bootstrap, gravatar avatar plugin with email notification. Add shortcode [mideal-faq-form] on your page to set the form question. Add shortcode [mideal-faq] on your page to implement the question list.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/mideal-faq` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Question ansver screen to configure the plugin
4. Use shorcode [mideal-faq-form] on a page for add form.
5. Use shorcode [mideal-faq] on a page for add list Question and answer.
